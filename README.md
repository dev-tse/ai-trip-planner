# 🗺 AI Trip Planner — Telegram-бот экскурсовод

AI-агент в Telegram, который планирует маршруты по городам на основе реальных данных OpenStreetMap. Пользователь описывает свою поездку в свободной форме — бот сам решает, какие места искать, считает расстояния между ними и составляет маршрут по дням.

## Что умеет

- Принимает свободный запрос ("2 дня в Алматы с семьёй, без машины")
- Самостоятельно ищет достопримечательности через OpenStreetMap (Nominatim API)
- Считает реальные расстояния между местами (haversine формула)
- Составляет маршрут по дням с учётом транспорта и состава группы
- Не рекомендует заведения, несовместимые с халяльным образом жизни
- Работает по любым городам мира, не только Казахстану

## Стек

- Python 3.9
- Groq API (openai/gpt-oss-120b) — LLM с tool calling
- OpenStreetMap Nominatim API — поиск мест и координат (бесплатно, без ключа)
- haversine — расчёт расстояний между координатами
- pyTelegramBotAPI — Telegram бот
- Деплой: Railway

## Архитектура — Agentic Loop с двумя инструментами

```
Сообщение пользователя
    → Модель решает: нужен инструмент?
        → search_places(query) — ищет место в OpenStreetMap, возвращает name/lat/lon
        → get_distance(place1, place2) — считает расстояние между двумя точками
        → Модель видит результат, решает: достаточно или искать ещё?
    → Финальный ответ с маршрутом
```

Модель может вызывать инструменты несколько раз подряд — например, переформулировать поисковый запрос если первый вернул нерелевантный результат (наблюдаемое поведение в продакшене).

## Известные ограничения

- **Качество данных:** Nominatim возвращает только координаты и название — часы работы, описания, отзывы модель добавляет из своих общих знаний, что может быть устаревшим. Для продакшена нужен Google Places API.
- **Галлюцинации:** модель может дополнять информацию о местах своими знаниями — частично решено через инструкцию в `description` инструмента и system prompt.
- **Время ответа:** агент делает несколько последовательных запросов (поиск → расчёт расстояний → финальный ответ), среднее время ответа 1-3 минуты.

## Запуск локально

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Создать `.env` файл:
```
GROQ_API_KEY=your_key
TELEGRAM_BOT_TOKEN=your_token
```

Запустить:
```bash
python ai_planner.py
```

## Пример использования

```
Пользователь: Я буду 2 дня в Алматы, куда сходить?

Бот: День 1: Кок-Тобе (канатная дорога, виды на город)...
     День 2: Медеу → Шымбулак (горнолыжный курорт)...
     Расстояние от центра до Медеу: 14.3 км
```
